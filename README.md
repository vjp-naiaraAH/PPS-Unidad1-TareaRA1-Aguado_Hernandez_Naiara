## Descripción de la tarea

Tarea obligatoria de la Unidad 1 del módulo **Puesta en Producción Segura**.  
Se ha analizado, depurado, probado y documentado una aplicación Python que simula un túnel de lavado de coches, cumpliendo los resultados de aprendizaje RA1 (a, b, c, d, e).

### Objetivos cumplidos

- Documentación detallada del código mediante Jupyter Notebook
- Ejecución y depuración en VS Code (terminal + depurador visual)
- Diseño y ejecución de 14 pruebas unitarias con `unittest`
- Verificación de reglas de negocio (precios, fases, excepciones)
- Ejecución en entorno aislado (sandbox con Firejail)
- Reflexión sobre seguridad en lenguajes de programación
